\doxysubsection{Detailed Description}
Include \texorpdfstring{$<$}{<}\mbox{\hyperlink{matrix__transform_8hpp}{glm/gtc/matrix\+\_\+transform.\+hpp}}\texorpdfstring{$>$}{>} to use the features of this extension.

Defines functions that generate common transformation matrices.

The matrices generated by this extension use standard Open\+GL fixed-\/function conventions. For example, the set\+Look\+At function generates a transform from world space into the specific eye space that the projective matrix functions (perspective, ortho, etc) are designed to expect. The Open\+GL compatibility specifications defines the particular layout of this eye space.
