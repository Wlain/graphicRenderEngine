{\footnotesize\ttfamily GLFWAPI int glfw\+Get\+Error (\begin{DoxyParamCaption}\item[{const char $\ast$$\ast$}]{description }\end{DoxyParamCaption})}



Returns and clears the last error for the calling thread. 

This function returns and clears the \mbox{\hyperlink{group__errors}{error code}} of the last error that occurred on the calling thread, and optionally a UTF-\/8 encoded human-\/readable description of it. If no error has occurred since the last call, it returns \mbox{\hyperlink{group__errors_gafa30deee5db4d69c4c93d116ed87dbf4}{GLFW\+\_\+\+NO\+\_\+\+ERROR}} (zero) and the description pointer is set to {\ttfamily NULL}.
