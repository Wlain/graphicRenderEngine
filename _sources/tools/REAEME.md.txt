# README

### 框架介绍
[Sphinx](https://www.sphinx-doc.org/en/master/) 是 Python 社区编写和使用的文档生成器。它是用 Python 编写的，也可以在其他环境中使用。

### 关于本文档
本文档是由[rst](https://www.sphinx-doc.org/en/master/usage/restructuredtext/basics.html) , [md](https://www.markdownguide.org/basic-syntax.html) 文件生成, 具体可以看[教程](https://docs.readthedocs.io/en/stable/tutorial/)

### 环境配置
macOS:
``` shell
# sphinx-doc工具
brew install sphinx-doc

# python插件依赖
pip install sphinx_rtd_size
pip install sphinx_markdown_tables
pip install myst_parser
```
